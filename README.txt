Wordpressをインストール
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
①「LAA0676594-template.sql」をエディタで開く
②「http://asuki.co.jp/tp01 」を、サイトアドレスに一括置換→保存
（※アドレス末尾に"/"を付けないよう注意）
③保存したSQLを、データベースにインポート
④wp-config内「WordPress データベーステーブルの接頭辞（$table_prefix）」を「wp1_」に変更
⑤サイトアドレス/wp-adminにアクセスし、指示に従ってインストールを完了する
（※インストール完了後に必要となるログインIDとパスワードはあすき社員には配布済みです。わからない場合は連絡ください。）

メニュー（商品）追加
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
①ダッシュボード＞メニューアイテム＞カテゴリーにて、新規カテゴリーを追加。
②ダッシュボード＞メニューカテゴリー＞新規追加にて空欄を埋める。「カテゴリースラッグ」は、①で作成したカテゴリーのスラッグ。
③ダッシュボード＞メニューアイテム＞新規追加にて、空欄を埋める。右側「カテゴリー」にて、①②で作成した該当するカテゴリーを選択すること。

【小カテゴリー】（※「ドリンク」カテゴリー内の「アルコール」設定の場合）
①ダッシュボード＞メニューアイテム＞カテゴリー＞新規追加にて、「アルコール」追加。親に「ドリンク」を選択する。
②ダッシュボード＞メニューアイテムの中から、小カテゴリー「アルコール」内で一番上にくるアイテムを選択（※この場合だと「宮内会長のハイボール」）
③「宮内会長のハイボール」編集画面＞カテゴリー一覧＞「ドリンクメニュー」と「アルコール」にチェックして保存。
④実際のページにて、「宮内会長のハイボール」の上に「ALCOHOL（アルコール）」というタイトルが追加されていることを確認する。


地図（※フッターのGoogle Map設定です）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
APIキー取得方法
①https://developers.google.com/maps/web/ にアクセス
②ページ右上にある「キーを取得」をクリック
③「アプリケーションを登録するプロジェクトの選択」という画面が表示されますので、「新しいプロジェクトを作成」のまま「続行」
名前は、ウェブサイト名などの名前でOKです。「HTTPリファラー」選択し、使用したいドメインを入力します。
④https://console.developers.google.com/apis/library にて、
プロジェクトを選択＞認証情報＞プロジェクト名＞「この HTTP リファラー（ウェブサイト）からのリクエストを受け入れる 」に、GoogleMapを使用するページ全てのアドレスを追加する。
「作成」ボタンを押して、「保存」をクリックすると、GoogleマップのAPIキーの取得が完了します。
⑤サイトの所有者を確認。https://www.google.com/webmasters/verification/ にアクセス
⑥プロパティを追加→使用したいドメインを入力します。→「続行」
⑦生成されたhtmlファイルをダウンロードし、ドメイン以下にアップロード→「確認」を押し、所有者の確認を完了します。
⑧Google側の認証に少し時間がかかるのでしばらく待ってから表示を確認する。※５～１０分
☆緯度・経度はここで正しい数値を調べましょう→http://user.numazu-ct.ac.jp/~tsato/webmap/sphere/coordinates/advanced.html
